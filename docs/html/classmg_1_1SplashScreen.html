Detailed Description

template<typename T>
class mg::SplashScreen< T >

Sequence of rotating screens

Template Parameters
Ttype of splash screen page

Initialized in READY state and wait to display() call for start screen rotation. Method tick() should be call periodically (usually inside loop()) to nonbloking screen rotation after screen display interval.

◆ display()

template<typename T >
const Screen< T > * mg::SplashScreen< T >::display ( )
inline

Starts splash screen from frist screen.

Changes splash screen status to mg::Status::DISPLAYING and store displaying screen pointer for first screen, its index = 0 and current millis() value.
For screen changing after time intervals method tick() should be periodically called. display() call is equivalent to display(0).

Returns
first screen pointer or null if screen not found.
